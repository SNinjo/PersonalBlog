Pyenv

版本控制

  • 列出所有已安裝的版本 
    pyenv versions
    shell
  • 列出所有可安裝的版本 
    pyenv install -l
    shell
  • 安裝指定版本 
    pyenv install -v <version>
    shell
  • 切換版本
    作用於整個系統 
    pyenv global <version>
    shell
    作用於這個資料夾 
    pyenv local <version>
    shell
    作用於這個 CLI 
    pyenv shell <version>
    shell

Venv

虛擬環境 (獨立各專案的套件)

  • 建立新的虛擬環境 
    python -m venv <name>
    shell
  • 使用指定的環境
    Windows 指令 
    <name>\Scripts\activate.bat
    shell
    Unix 或 MacOS 指令 
    source <name>/bin/activate
    shell
  • 停止使用所有環境 
    deactivate
    shell

Pip

套件管理

  • 安裝指定套件 
    pip install <package>
    shell
  • 安裝所有陳列在 requirement.txt 的套件 
    pip install -r requirements.txt
    shell
  • 輸出所有套件至 requirement.txt 
    pip freeze > requirements.txt
    shell
