Root 權限
  • sudo (superuser do): 輸入自己的密碼取得權限 
    sudo <command>
    shell
  • su (switch user): 輸入 root 的密碼取得權限並開一個新的 shell 
    su - \
        -c "<command>"
    shell
  • 結合兩者: 輸入自己的密碼取得權限並開一個新的 shell 
    sudo su -
    shell

ssh

secure shell (protocol)

一種加密的網路傳輸協定，可在不安全的網路中為網路服務提供安全的傳輸環境

ssh <user>@<host|ip> \
    -p <port>
shell

檔案系統
  • 創建檔案 
    touch <name>
    shell
  • 創建資料夾 
    mkdir <name>
    shell
  • 將檔案或是資料夾改名 
    mv <old name> <new name>
    shell
  • 移動檔案或是資料夾 
    mv <old path> <new path>
    shell
  • 複製檔案 
    cp <old path> <new path>
    shell
  • 複製資料夾 
    cp -r <old path> <new path>
    shell
  • 刪除檔案或是空資料夾 
    rm <name>
    shell
  • 刪除內含檔案的資料夾 
    rm -r <name>
    shell

lsof

list open files

lsof \
    -i (tcp|udp)?:<port>? \
    -p <process ID> \
    -u <user name> \
    -c <command/program name>

watch

用來重複執行固定指令，並且監看輸出訊息的工具。

  • 標注不一樣的地方 
    watch -d <command>
    shell
  • 設置執行間隔 
    watch -n <seconds> <command>
    shell
  • 隱藏標頭 
    watch -t <command>
    shell

硬碟
  • df (disk free): 列出整個系統硬碟的使用情況, 參數「-lh」使得輸出格式更容易閱讀 
    df -lh
    shell
  • du (disk usage): 列出資料夾空間的使用情況
    顯示資料夾空間的總大小 
    du -sh
    shell
  • fdisk (format disk): 互動模式的磁碟分割區工具 列出當前操作硬碟的分區情況 
    fdisk -l
    shell
  • 將裝置掛載至檔案系統的指定資料夾上，使檔案系統可以存取裝置上的檔案如同資料夾一樣 
    mount <device name> <directory path>
    shell
    卸載裝置 
    umount <directory path>
    shell

systemctl

control the systemd system and service manager

  • 列出服務
    • 啟動的服務
    systemctl
    shell
    • 未啟動的服務
    systemctl list-units --all --state=inactive
    shell
    • 所有服務
    systemctl list-units --all
    shell
  • 啟動服務 
    systemctl start <service>
    shell
  • 停止服務 
    systemctl stop <service>
    shell
  • 重啟服務 
    systemctl restart <service>
    shell
  • 使該服務自動啟動 
    systemctl enable <service>
    shell
  • 使該服務手動啟動 
    systemctl disable <service>
    shell

curl

client URL

在不同協定下，透過 URL 發送請求、傳輸資料的工具。

curl <URL> \
    -X <method written in all caps> \
    -H "<header>" \
    -d '<data>' \
    -F "<form data>" \
    --cookie "<cookie>" \
    --user-agent "<user agent>"
shell

GET JSON 範例:

curl https://www.google.com \
    -X GET \
    -H "Content-Type: application/json" \
    -d '{"status" : false, "name": "SNinjo"}' \
    --cookie "name=SNinjo" \
    --user-agent "Mozilla/5.0 (compatible; MSIE 5.01; Windows NT 5.0)"
shell

Form Data 範例 (「@」代表檔案路徑):

curl https://www.google.com \
    -X POST \
    -H "Content-Type: multipart/form-data" \
    -F "name=SNinjo&file=@img/SNinjo.png"
shell

Bash

  • 清除所有輸出 (鍵盤快捷鍵是「Ctrl + L」)

    clear
    shell

  • 顯示命令歷史紀錄

    history
    shell

  • 輸出 <string> 並換行

    echo <string>
    shell

  • 延遲執行程式一段指定時間

    sleep <second>
    shell

  • 串接指令用「&&」

    <command> && <command>
    shell

  • 只顯示包含搜尋字的輸出

    <command> | grep <search word>
    shell

    在空白前面加上「\」, 範例:

    echo "12 3" | grep 12\ 3
    shell

  • 只顯示最後 <number> 行的輸出

    <command> | tail -<number>
    shell

  • while 迴圈

    while <boolean>; do <command>; done
    shell

    舉例來說, 每秒 ping <IP> 且顯示時間

    while true; do date && ping 61.222.241.174 -c 1 && sleep 1 && echo; done
    shell
