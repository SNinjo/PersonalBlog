容器
  • 列出所有容器 
    docker ps
    shell
  • 將映像檔移至容器執行 
    docker run \
        --name <container name> \
        -p <outer port>:<inner port> \
        -v <outer volume path>:<inner volume path> \
        <image id|repository>:<tag>?
    shell
  • 暫停容器 
    docker stop <image id|repository>:<tag>?
    shell
  • 刪除容器 
    docker rm <image id|repository>:<tag>?
    shell

映像檔
  • 列出所有映像檔 
    docker image ls
    shell
  • 標註映像檔 
    docker tag <image id|repository>:<tag>? <repository>:<tag>?
    shell
  • 刪除映像檔 
    docker image rm <image id|repository>
    shell

註冊伺服器

每個註冊伺服器對於 <repository> 都有不同的定義, 且它也決定映像檔將更新到何處。

  1. Docker hub:
    <repository> = <user name>/<repository name>
  2. Harbor:
    <repository> = <domain>/<project>/<repository name>
  • 登入 docker hub 
    docker login
    shell
  • 登入 private docker registry 
    docker login <domain>
    shell
  • 推送映像檔上去伺服器 
    docker push <repository>:<tag>?
    shell
  • 拉取映像檔下來本地端 
    docker pull <repository>:<tag>?
    shell

其他
  • 刪除所有的映像檔、容器、快取 
    docker system prune -a
    shell
