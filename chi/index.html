常用快捷鍵

ctrl + T = 開新分頁

ctrl + W = 關閉分頁 (若是最後一個分頁則關閉視窗)

ctrl + N = 開新的視窗 (保留原本的帳號設定)

ctrl + shift + N = 開無痕視窗

ctrl + Tab = 向右切換分頁

ctrl + shift + Tab = 向左切換分頁

Github 教學

1. 先將 branch 切到專屬你編輯的 chi

2. 點選 Content 資料夾 (文章都放在這)

3. 點選要編輯的文章

4. 點選編輯按鈕

5. 改好文章後捲至下方, 填寫 changes 並按下 commit, 口以寫個 feedback 給我~

Trello 教學

1. 新增卡片

2. 點擊欲編輯的卡片

3. 可以留言紀錄筆記，或是加入標籤

4. 更新文章狀態 (滑鼠左鍵點住卡片，拖移至另一個列表再放開)
