「vi」是常用於 UNIX 的文字編輯器，而「vim」是其改良版 (為「Vi IMproved」的縮寫)。
查看更多介紹

vim <file name?>
shell

Vim 有多種模式，主要常見的分別為以下四種。

命令模式
  • 切換模式
    1. 至插入模式 
      i
      command
    2. 至命令行模式 
      :
      command
    3. 至視覺模式 
      v
      command
  • 恢復之前的修改 
    u
    command
  • 複製游標所在的這一行 
    yy
    command
    貼上至游標所在的上一行
    (大寫) 
    P
    command
    貼上至游標所在的下一行
    (小寫) 
    p
    command
  • 刪除
    • 游標所在的下一個字元 
      x
      command
    • 游標所在的這一行 
      dd
      command

插入模式

編輯檔案內容

  • 切換至命令模式: 按下 Esc

命令行模式
  • 切換至命令模式: 按下 Esc
  • 開啟檔案 (edit) 
    :e <file path?>
    command
    重新開啟檔案並捨棄尚未儲存的修改 
    :e!
    command
  • 關閉檔案 (quit) 
    :q
    command
    關閉檔案並捨棄尚未儲存的修改 
    :q!
    command
  • 儲存檔案 (write) 
    :w
    command
    強制儲存 
    :w!
    command
    指定名稱另存檔案 
    :w <file path>
    command
    儲存並關閉檔案 
    :wq
    command
  • 將游標移至第 <line number> 行 
    :<line number>
    command
    移至最後一行 
    :$
    command

視覺模式
  • 切換至命令模式: 按下 Esc
