Variable

Variable is a named storage space that can store a value. It created the moment we first assign a value to it. Before we use any variable, it need to be declared.
For example, we declare "x" and assign 10 to a variable called "y".

x        # declare x
y = 10   # declare y and assign 10 to y
python

There are common variable type.

  • Boolean
    is only "True" or "False", usually used to the judgement. 
    x = True
    y = False
    python
  • Number
    is called numeric in Python, which includes "integer" and "floating-point number". 
    x = 10   # integer
    y = 1.5  # float
    python
  • String
    is the text. Following are the three types of string. 
    x = "I'm string."
    y = 'I am string, too.'
    z = '''I am also string'''  # It can contain newline character "\n".
    python
  • Object
    has the attribute and method, which can be called by key, such as key-value pair. In Python, we can define it with dictionary or curly braces. 
    x = dict(  id=1, name='sninjo'  )  # dictionary
    y = {  id: 2, name: 'jo'  }  # curly braces
    x['id']  # 1
    y["name"] = 'alan'  # assign 'alan' to the attribute with key "name"
    python
  • Array
    is a basic data structure used to store values ​​in contiguous storages. The index is used to pair the elements, starting from 0. We define it with brackets. 
    x = [<object>, <object>, ...]
    x[<index>]
    python
    x = [1, 2, 3]
    x[0]  # 1
    x[1]  # 2
    x[2] = 10  # assign 10 to the element with index "2"
    python

Function

Function is a block of code which only runs when it is called. In python, we define the block of code with indentation instead of curly braces, and "return" is used to leave it and send the result.
By the way, The parameter is the variable listed inside the parentheses in the function definition. The argument is the value that is sent to corresponding parameter in the function when it is called.

def <name>(<parameter>, <parameter>, ...):
    <action>                         # inside the function
<name>(<argument>, <argument>, ...)  # call the function with arguments
python
def sum(x, y):
    return x + y
sum(1, 2)
python

There are common function.

  • print()
    shows the string on the terminal. 
    print('Hello World')
    x = True
    print(x)
    python
    Hello World
    True
    output
  • type()
    returns the type of the object. 
    print(type('Hello World'))
    x = True
    print(type(x))
    python
    <class 'str'>
    <class 'bool'>
    output
  • len()
    returns the length of the object. 
    print(len('Hello World'))
    x = [1, 2, 3]
    print(len(x))
    python
    11
    3
    output
  • abs()
    returns the absolute value of the specified number. 
    print(abs(10))
    x = -5
    print(abs(x))
    python
    10
    5
    output
  • range()
    returns a sequence of numbers, such as range(min, max, step), starting from "min", and increments by "step", and stops before "max". 
    range(0, 5, 1)   # 0, 1, 2, 3, 4
    range(9, 4, -2)  # 9, 7, 5
    python

Loop

Loop is used to repeat a block of code. In Python, we use indentation instead of curly braces to define the block of code. Furthermore, This is known as iteration, which allows us to "write code once" and "execute many times."

There are two types of loop.

  • while 
    while <condition>:
        <action>
    python
    x = True
    while (x):
        print('while')
        x = False
    python
    while
    output
  • for
    Array and range() are the iterable object. In for loop, each action can get one element from the iterable object in order. 
    for <object> in <iterable object>:
        <action>
    python
    for x in [0, 1, 2]:
        print(x)
    for x in range(3, 5, 1):
        print(x)
    python
    0
    1
    2
    3
    4
    output

There are two useful statement.

  • break
    exits the loop, which means leaving the block of code. 
    for x in [0, 1, 2]:
        print(x)
        break
        print('Hi')
    print('end')
    python
    0
    end
    output
  • continue
    ends the current iteration in a loop, and continues to the next iteration. 
    for x in [0, 1, 2]:
        print(x)
        continue
        print('Hi')
    print('end')
    python
    0
    1
    2
    end
    output

Math
  • Addition
    add two numbers together 
    5 + 2   # 7
    python
  • Subtraction
    subtract the right number from the left 
    5 - 2   # 3
    python
  • Multiplication
    multiply two numbers together 
    5 * 2   # 10

    divide the left number by the right 
    5 / 2   # 2.5

    raise a base number to the exponent power, that is, the base number multiplied by itself, exponent times. 
    5 ** 2  # 25
    python

  • Single-line 
    # I'm a comment.
    x = 10  # I'm also a comment.
    python
  • Multi-line
    In fact, this is a string, which can contain newline character "\n". 
    '''
    first line comment
    second line comment
    '''
    python
