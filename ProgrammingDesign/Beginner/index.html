Before starting to read this chapter, we need to do the following things. By the way, we choose Python as our teaching programming language.

Install Python
  1. Open Python Download Page and click latest version.
  2. Tick all options below and click "Install Now", PATH is an environment variable that can execute Python commands in the CLI.
  3. Press win + R and enter "CMD" to open CLI in Windows.
  4. enter "python" to check if Python is downloaded.

Choose IDE

IDE (short for Integrated Development Environment) is a text editor with compiler, debugger, and a lot of plugins, and we can easily and conveniently design any program with it.

  1. Online Console If you don't want to install any software, you can just use Online Console in Python official website. However, it is very simple and isn't recommended to use it.

  2. IDLE IDLE (short for Integrated Development and Learning Environment) is IDE for Python, which has been bundled with the default implementation of the language.

  3. VSCode VSCode (short for Visual Studio Code) is most popular IDE recently, because of its simplicity, lightweight, and extensibility.
