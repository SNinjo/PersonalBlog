初學者

在開始之前，讓我們先來完成一些必要的配置。此外，這個章節將以 Python 作為示範的程式語言。

安裝 Python

  1. 開啟 Python 下載頁面 並安裝最新版本
  2. 勾選下方所有的選項並點擊「Install Now」, PATH 是讓 CLI 能認得這個程式語言相關配置的環境變數
  3. 鍵盤點擊 win + R 開啟「執行」視窗後輸入「cmd」以開啟 Windows 的 CLI
  4. 輸入「python」確認 Python 是否成功下載

選擇 IDE

整合開發環境 (Integrated Development Environment，簡稱 IDE) 是一個附帶編譯器、除錯器與許多套件的文字編輯器。有了它的幫助，我們可以方便且快速地開發各式各樣的系統。

  1. Online Console

如果你不想安裝任何的軟體在電腦上，那就可以選擇 Python 官方網站上提供的 Online Console。 然而它所提供的功能非常的簡易，若是想開發有點規模的系統則不建議使用。

  1. IDLE

IDLE (Integrated Development and Learning Environment) 是 Python 內建附帶的 IDE，相較於 Online Console 提供較多的功能且不會延遲。

  1. VSCode

VSCode (Visual Studio Code) 具有輕量、易上手且擴展性高的特性，目前 VSCode 是最普及的 IDE。
